# Export Transcriptions

AIRecorder lets you export your transcriptions in various formats to suit different needs.

## Available Formats

### Plain Text (TXT)

The simplest format. Copies only the transcription text without any formatting.

### Markdown (MD)

Structured format that preserves:
- Transcription headers
- Line breaks
- Formatting
- List format

Ideal for editing in apps like Obsidian, Notion, or VS Code.

### PDF

Formatted document ready to print or share. Includes:
- Transcription title
- Date and duration
- Full text with timestamps
- AI analysis (summary, tasks, key points)

### SRT (Subtitles)

Subtitle format with timestamps. Useful for:
- Creating subtitles for videos
- Syncing with video players

## Export Options

### Include Timestamps

Enable this option to add timestamps next to each transcribed segment. Example:

```
[00:00:15] Hello everyone, I'll start by presenting the project.
[00:00:42] Thank you, now I'll explain the technical details.
```

### Include Speaker Names

If you have diarization enabled, you can export with assigned names for each speaker:

```
John: Good morning, thanks for attending the meeting.
Maria: It's a pleasure to be here presenting the results.
```

### Include AI Analysis

You can choose to include in the export:
- Meeting summary
- Key points
- Pending tasks

## How to Export

1. Open the transcription you want to export
2. Click the **Export** button (download icon)
3. Select the desired format
4. Configure options (timestamps, speakers, analysis)
5. Choose the location to save the file

::: tip
You can quickly export using the keyboard shortcut `Cmd/Ctrl + E` while viewing a transcription.
:::

## Use Cases

| Format | Best for |
|--------|----------|
| TXT | Copy and paste into other documents |
| MD | Edit in note apps or wikis |
| PDF | Share with people who don't use code apps |
| SRT | Create subtitles for videos |
