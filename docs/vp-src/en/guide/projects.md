# Projects

Projects allow you to **group and organize** your transcriptions by context, making it easier to search and manage your content.

## Create a Project

1. Go to the **Projects** section in the sidebar
2. Click **+ New Project**
3. Enter a descriptive name (e.g., "Work Meetings", "Physics Classes")
4. Optional: Add a description

## Assign Recordings to a Project

### From the Recording

1. Open the transcription you want to organize
2. In the details panel, look for the **Project** field
3. Select the project the recording belongs to

### Change the Project

1. Go to the transcription list
2. Right-click on the transcription
3. Select **Move to** and choose the new project

## Filter by Project

In the main view, you can filter transcriptions by project:

- **All**: Shows all transcriptions
- **No project**: Shows unassigned transcriptions
- **[Project name]**: Shows only that project's recordings

## Benefits of Using Projects

| Benefit | Description |
|---------|-------------|
| **Organization** | Groups related content |
| **Quick search** | Filter by context |
| **Context** | Keeps the purpose of each recording clear |
| **History** | Track topics by project |

## Example Uses

- **Work**: Team meetings, client calls, standups
- **Study**: Classes, seminars, lectures
- **Personal**: Ideas, family meetings, interviews
- **Projects**: Design, development, research
