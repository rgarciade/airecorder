# Exportar Transcripciones

AIRecorder te permite exportar tus transcripciones en varios formatos para adaptarse a diferentes necesidades.

## Formatos Disponibles

### Texto Plano (TXT)

El formato más simple. Copia solo el texto de la transcripción sin ningún formato.

### Markdown (MD)

Formato estructurado que conserva:
- Encabezados de la transcripción
- Saltos de línea
- Né¡umtabs
- Formato de lista

Ideal para editar en apps como Obsidian, Notion o VS Code.

### PDF

Documento formateado listas para imprimir o compartir. Incluye:
- Título de la transcripción
- Fecha y duración
- Texto completo con timestamps
- Análisis de IA (resumen, tareas, puntos clave)

### SRT (Subtitles)

Formato de subtítulos con timestamps. Útil para:
- Crear subtítulos para videos
- Sincronización con players de video

## Opciones de Exportación

### Incluir Timestamps

Activa esta opción para añadir marcas de tiempo junto a cada segmento transcrito. Ejemplo:

```
[00:00:15] Hola a todos, empezaré presentando el proyecto.
[00:00:42] Gracias, ahora voy a explicar los detalles técnicos.
```

### Incluir Nombres de Hablantes

Si tienes la diarización activada, puedes exportar con los nombres asignados a cada hablante:

```
Juan: Buenos días, gracias por asistir a la reunión.
María: Es un placer estar aquí presentando los resultados.
```

### Incluir Análisis de IA

Puedes elegir incluir en la exportación:
- Resumen de la reunión
- Puntos clave
- Tareas pendientes

## Cómo Exportar

1. Abre la transcripción que deseas exportar
2. Haz clic en el botón **Exportar** (icono de descarga)
3. Selecciona el formato deseado
4. Configura las opciones (timestamps, hablantes, análisis)
5. Elige la ubicación para guardar el archivo

::: tip
Puedes exportar rápidamente usando el atajo de teclado `Cmd/Ctrl + E` mientras visualizas una transcripción.
:::

## Casos de Uso

| Formato | Mejor para |
|---------|------------|
| TXT | Copiar y pegar en otros documentos |
| MD | Editar en apps de notas o wikis |
| PDF | Compartir con personas que no usan apps de código |
| SRT | Crear subtítulos para videos |
