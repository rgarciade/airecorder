# Schema / Mind-Map

Each recording can have a **schema** (mind-map) generated by AI. It's a structured summary with thematic branches and key points anchored to the exact second of the audio.

You can click any node with a timestamp to jump to that moment in the audio.

---

## Outline View

The **Outline** view shows the schema as a collapsible hierarchical list. Each branch has:

- **Branch title**: the main topic
- **Nested points**: details, decisions, dates, owners
- **Timestamp**: shown as `(MM:SS)` — click the ▶ button to seek the audio

Branches can be collapsed/expanded individually.

---

## Mind-Map View

The **Mind-Map** view renders the schema as an interactive mind map using [markmap](https://markmap.js.org/):

- **Zoom/Pan**: navigate with mouse or touch gestures
- **Timestamp nodes**: cursor changes to pointer — click to seek audio
- **Colors**: each depth level has a distinct color for readability

---

## Generating the Schema

1. Open a recording that already has a transcription
2. Go to the **"Outline"** tab (third position, after Overview)
3. Click **"Generate outline"**
4. AI analyzes the transcription segment by segment and generates branches with precise timestamps

You can **regenerate** the schema at any time if it didn't turn out as expected.

### Requirements

- The recording must have a **segmented transcription** (not just plain text)
- A **configured AI provider** is required (local or cloud)
- If a **detailed summary** already exists, it's used as additional context to improve quality

---

## Auto-generation

In **Settings > General > Transcription** you can enable **"Auto-Generate Schema"**.

When enabled, after the initial AI analysis completes (summary, tasks, key points), the schema is generated automatically if it doesn't exist yet.

---

## Export

### Markdown (.md)

Generates a nested outline document with timestamps on each key point. Ideal for sharing or integrating with other tools.

### PNG (.png)

Exports the visual mind-map as a PNG image. The capture is done natively via Electron (no external libraries required).

::: warning
To export PNG, you must be in **Mind-Map view** (not Outline).
:::

---

## Schema in AI Chat

If RAG chat is active (recording mode), the schema is injected into the model's context as additional information. This also happens in project chats when recordings with schemas exist.

This allows the AI to answer questions about the **overall structure** of the meeting even if the relevant fragment isn't in the top semantic search results.

---

## Internal Structure

The schema is saved as JSON at `<recordings>/<folderName>/analysis/recording_schema.json`:

```json
{
  "branches": [
    {
      "title": "Specific topic",
      "start": null,
      "children": [
        { "label": "Point or decision", "start": 192.4, "children": [] }
      ]
    }
  ]
}
```

Each node supports recursive nesting with no artificial limit. Legacy formats (`items[]`) are still read for backward compatibility.
