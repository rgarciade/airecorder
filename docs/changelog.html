v0.6.1 Junio 2026 main
Wiki por Proyectos, Widget Flotante y Correcciones
Nuevas funciones y correcciones
📓
Wiki por proyectos
Nueva sección Wiki dentro de cada proyecto. Permite crear, editar y organizar páginas de documentación vinculadas al proyecto.
🎛️
Widget flotante de grabación
Nuevo widget flotante que muestra el estado de la grabación activa y permite controlarla sin necesitar abrir la ventana principal.
💬
Importar conversaciones
Soporte para pegar o importar conversaciones de chat (texto plano, archivos, o conversaciones analizadas por IA) directamente en una grabación.
✏️
Editor de plantillas
Nuevo editor y selector modal de plantillas para personalizar los prompts usados en el análisis y chat con IA.
🗣️
Mejoras en gestión de hablantes
Nuevo selector de hablantes, hooks dedicados para lista y fusión, y resolución correcta de segmentos heredados del sistema de diarización.
🐛
Correcciones de estabilidad
Se corrigen 4 bugs: el monitor de micrófono fallaba al compilar su binario en la app distribuida; settings.json podía corromperse por escrituras concurrentes; el historial de chat fallaba para grabaciones con ID numérico; stale closure en el widget flotante al detener la grabación.
v0.5.1 Abril 2026 main
Hablantes, Auto-análisis IA y Prompts de Experto
Nuevas funciones
🗣️
Identificación de Hablantes
Nueva página de Hablantes con gestión completa: diarización automática para identificar quién habla en cada segmento, umbral de similitud configurable, fusión de hablantes con autocompletado, y posibilidad de desvincular un hablante de una grabación.
🧑‍💼
Prompts de Experto
Nuevos perfiles de experto para el chat con IA: Desarrollador de Software y Psicólogo. Cada perfil ajusta el comportamiento del asistente para ofrecer respuestas especializadas según el contexto de la grabación. También se añade un asistente general para conversaciones sin contexto específico.
🤖
Auto-análisis IA configurable
Activá el auto-análisis desde los ajustes y, al finalizar cada grabación, la IA la analizará automáticamente: genera el resumen, extrae participantes y tareas sin intervención manual. Podés añadir instrucciones personalizadas por grabación para guiar el análisis. También se puede configurar la auto-transcripción para que el proceso completo ocurra de forma desatendida.
💬
Importa conversaciones de chat con IA
Importa archivos de conversaciones o chats analizados por IA y tratalos como cualquier grabación: resumen automático, extracción de tareas, búsqueda semántica y chat interactivo sobre el contenido.
🎙️
Monitor de Micrófono
Nuevo monitor de micrófono en segundo plano: cuando se detecta actividad de audio, AIRecorder lanza una notificación para avisarte. Vos decidís si iniciás la grabación o la ignorás. Ideal para no perder conversaciones inesperadas.
⚙️
Ajustes Generales Reorganizados
La sección de Ajustes cuenta ahora con una pestaña General completamente rediseñada con secciones para Apariencia, Audio, Proyectos, Almacenamiento, Sistema, Permisos y Transcripción.
✨
Mejoras generales
Soporte para ReactMarkdown en resúmenes, conversión de LaTeX a Unicode en el chat, acceso a la wiki desde la barra lateral y ajustes, y soporte para donaciones en Ko-fi.
v0.4.1 Marzo 2026 main
Versión para Windows y Mejoras de IA
Nuevas funciones
💻
Versión para Windows
¡AIRecorder ahora es compatible con Windows! Se ha creado una build específica para Windows (x64) y se han realizado los ajustes de compatibilidad necesarios. La web ahora muestra el botón de descarga correcto según tu sistema operativo.
🌙
Modo Oscuro Completo y Mejoras de UI
Se ha implementado un modo oscuro completo en toda la aplicación para una mejor experiencia visual en condiciones de poca luz. Además, se han realizado varias mejoras de contraste y estilos.
🧠
Mejoras en el Chat con IA
Ahora puedes navegar directamente a un momento específico de la transcripción desde un mensaje del chat. También se ha añadido un interruptor para alternar entre el modo RAG (búsqueda semántica) y el modo "Completo" que usa la transcripción entera como contexto.
🤖
Soporte para Modelos de Razonamiento Avanzado
Se ha mejorado el soporte para modelos de IA avanzados (Reasoning models), permitiendo análisis más profundos y respuestas más coherentes en tareas complejas. La re-indexación de la base de datos vectorial es ahora automática al cambiar de modelo.
v0.3.2 Marzo 2026 main
Correcciones críticas de estabilidad
Correcciones y mejoras
💥
Corrección de crash al transcribir
Se corrige un crash crítico en la versión compilada que bloqueaba la aplicación durante la transcripción. Era causado por una recursión infinita entre el handler de stderr de Python y el sistema de telemetría Sentry.
📄
Exportación mejorada con soporte Markdown
La exportación a PDF y DOCX ahora renderiza correctamente el formato Markdown del resumen: negritas, cursivas, listas y encabezados.
🤖
Mejor compatibilidad con modelos locales
Se mejora la generación de resúmenes, participantes y tareas con modelos locales (Ollama/LM Studio), concatenando correctamente el system prompt para modelos que no soportan el campo de sistema separado.
v0.3.1 Marzo 2026 feature/best-chat
Adjuntos con IA, System Prompts y Correcciones
Nuevas funciones
📎
Adjuntos como contexto para la IA
Adjunta imágenes, PDFs, documentos Word y Excel a cualquier grabación o proyecto. La IA los incorpora como contexto en el chat, con detección automática de soporte de visión para modelos compatibles.
🤖
Detección automática de soporte de visión
La app consulta la API de Hugging Face para detectar automáticamente si el modelo activo soporta imágenes, habilitando o restringiendo la selección de adjuntos de tipo imagen según corresponda.
🧠
System prompts y configuración dual de modelo
Nuevo sistema de prompts personalizados por contexto. Permite configurar modelos distintos para tareas generales y para el chat, ofreciendo más control sobre el comportamiento de la IA.
🔗
Botón "¿Puedo ejecutarlo?" integrado
Acceso directo desde la app a la herramienta canirun.com para comprobar si tu Mac puede ejecutar un modelo de IA local antes de descargarlo.
🐛
Correcciones de estabilidad y UI
Corrección del z-index del menú de adjuntos con React Portal, fixes en la selección de archivos según soporte de visión, mejoras en el botón de eliminar y el selector de archivos en proyectos.

v0.2.0 Marzo 2026 feature/ai-queue
Cola de IA, Multiidioma y Mejoras en Proveedores Locales
Nuevas funciones
🔄
Cola de procesamiento de IA
Genera resúmenes, actas y preguntas en segundo plano sin bloquear la app. Historial completo de tareas completadas con sus resultados.
📁
Proyectos con tablero Kanban
Agrupa grabaciones en proyectos. Cada proyecto tiene su propio tablero Kanban (Pendiente · En progreso · Completado) con todas las tareas generadas por la IA.
➕
Añadir tareas al proyecto desde la transcripción
Selecciona cualquier fragmento de texto en la transcripción y envíalo como tarea directamente al Kanban del proyecto.
🌐
Idioma de la interfaz e idioma de las respuestas de IA
Dos ajustes independientes: elige en qué idioma se muestra la interfaz (ES/EN) y en qué idioma la IA genera sus respuestas.
🔍
Búsqueda semántica (RAG)
Haz preguntas sobre tus grabaciones con búsqueda vectorial. Compatible con Ollama y LM Studio como backend de embeddings.
🎤
Importar transcripciones y audio
Importa transcripciones de Microsoft Teams en formato .docx o archivos de audio externos para transcribirlos localmente con Whisper.
📂
Rutas de almacenamiento configurables
Elige dónde guardar la base de datos y las grabaciones. Opción de migrar la BD existente al cambiar de ruta.
🧠
Ventana de contexto dinámica
El procesamiento se adapta automáticamente al contexto real del modelo cargado. Menos fragmentos innecesarios con modelos de 32k tokens o más.
⚠️
Errores de IA con detalle
Los errores del proveedor de IA se muestran en un modal con el mensaje real y sugerencias de solución.
